# Linux Bash Script – Decision Making Programs

This assignment demonstrates the use of conditional statements in Bash scripting to perform decision-based operations.

## Programs Included

### 1. Even or Odd Check
- Accepts a number from the user
- Uses modulus operator (%)
- Prints whether the number is Even or Odd

### 2. Largest of Three Numbers
- Accepts three numbers from the user
- Compares values using if-elif conditions
- Displays the largest number

### 3. Leap Year Check
- Accepts a year from the user
- Checks leap year conditions:
  - Divisible by 400  
  - OR divisible by 4 but not divisible by 100
- Prints whether it is a Leap Year or not

### 4. Bank Balance Calculation
- Accepts account balance and withdrawal amount
- Calculates tax based on withdrawal:
  - < 1500 → 3%
  - 1500–3000 → 4%
  - > 3000 → 5%
- Checks for insufficient balance
- Displays withdrawal amount, tax deducted, and remaining balance
